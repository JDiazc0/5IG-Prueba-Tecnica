### Instalación

- Python 3.10.11
- Libreria requests

  1. Clonar el repositorio y moverse a la carpeta `Task2/`.

  ```
  cd task2
  ```

  2. Crear entorno virtual y activar.

  ```
  python -m venv venv
  venv\Scripts\activate
  ```

  3. Instalar dependencias.

  ```
  pip install -r requirements.txt
  ```

  4. Ejecutar Script

  ```
  python script.py
  ```
